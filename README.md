# drone-path
Campus mapping using clustering and segmentation for drone navigation

This project explores clustering-based road detection and instance segmentation
for generating drone navigation paths from campus video data.

Pipeline:
1. Feature extraction using YOLOv8 backbone
2. Region clustering for road region detection
3. Instance segmentation for obstacle identification
4. Path extraction for drone navigation

   Step 1 done, working on step 2 by fine tuning for more accurate results
